# Breaking CSE_Squirrelmail CAPTCHA  
segmentation_cse.py is used to segment a given CSE CAPTCHA into 5 individul characters  
CSE & CSE_test contain individual characters after segmentation for training and testing respectively  
model.py is the structure of CNN that will be trained  
train_cse.py is used to train the model
mytrain_final.pt stores the learned model which will be used in testing
test_cse.py is used to test the trained model on test dataset
